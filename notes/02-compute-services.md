## Hour 1 – Amazon EC2

EC2 provides virtual servers in the cloud.

Key concepts:
- EC2 instance
- AMI
- Instance types
- Security groups

Example use case:
Running web applications on virtual servers.

**Architecture**
A typical AWS web architecture looks like this:
Internet
   ↓
Port 443
   ↓
Load Balancer
   ↓
EC2 Instances
   ↓
Database
